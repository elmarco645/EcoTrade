# EcoTrade Project Specification

## Folder Structure
- `.tmp/`: Temporary files and transient data.
- `backend/`: Express server logic, database configuration, and API routes.
- `directives/`: Project metadata, environment templates, and core instructions.
- `execution/`: Application entry points and boot logic.
- `frontend/`: React application, Vite configuration, and static assets.

## Core Features
- Secure Escrow System for second-hand items.
- Real-time Chat for buyer-seller negotiations.
- Virtual Wallet for managing funds.
- User Ratings and Trust system.
